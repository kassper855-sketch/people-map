# Карта людей

## Запуск на Windows

1. Установите Python 3.11+.
2. Откройте командную строку в папке проекта.
3. Выполните:

```bash
pip install -r requirements.txt
python app.py
```

4. Откройте `http://127.0.0.1:5000`.

Данные локально сохраняются в файле `people.db`.

## Важно для Render

Обычная файловая система Render может очищаться после перезапуска. Для постоянного хранения подключите PostgreSQL и добавьте переменную окружения `DATABASE_URL`, либо используйте Persistent Disk.
